## Architectural notes
### Astro 6 content API (S1)
glob loader with `generateId` stripping `/index.md`; entry render via
`render(entry)`. Draft filter applied in every getCollection call AND in
getStaticPaths (drafts have no routes at all).
### numpy convert() footgun (S1)
`ABCPolyBase.convert()` without `kind=` keeps the series kind — it does NOT
return power coefficients. Symptom if regressed: leading-coefficient rel
error of exactly 2^(m−1)−1 with R² agreeing exactly.
### Canvas plate (S1)
Viewer canvas always paints on `--rulebg` (dark plate) with `--bg` text —
intentional inversion, theme-stable.

## Things NOT to do
- Don't commit images that haven't been through strip_resize.py.
- Don't compare spectral stick heights across species (per-band norm; ASD
  relative intensities are source-dependent) — keep the UI caveat.
- Don't "fix" intensity strings into floats in the data files; qualifiers
  carry information. Parse leading numerics at draw time only.
- Don't reintroduce raw-monomial normal equations in calibration.
- Don't edit `--spectral-gradient` by hand; regenerate via the script.
- Don't mention visa/legal context anywhere in this public repo.
