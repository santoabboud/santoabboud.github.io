# santoabboud.github.io — v2

Astro static site. Datasheet design system (sand paper / carbon ink / burnt-amber,
CIE-computed spectral rule). All physics tools run client-side.

## Local dev
```bash
npm ci          # node ≥ 20 (CI uses 22)
npm run dev     # localhost:4321
npm run build   # production build → dist/
npm test        # calibration engine vs numpy references (see scripts/)
```

## First-time deploy of this rebuild  ⚠ replaces repo history
This tree is a pre-initialized git repo with a single fresh-root commit
(the old history is intentionally discarded — it contains GPS-tagged
images). Optional: archive the old repo first:
```bash
git clone --mirror git@github.com:santoabboud/santoabboud.github.io.git old-site-archive.git
```
Then:
```bash
cd site
git remote add origin git@github.com:santoabboud/santoabboud.github.io.git
git push -f origin main
```
Finally, on GitHub: **Settings → Pages → Build and deployment → Source:
"GitHub Actions"** (one-time). Every push to `main` then builds and deploys
via `.github/workflows/deploy.yml`.

## Adding a project
Create `src/content/projects/<slug>/index.md`:
```markdown
---
title: "Title"
description: "One-sentence card text."
category: lasers   # spectroscopy|microscopy|lasers|night-vision|cameras|batteries|art|simulation-software
date: 2026-06-15
cover: ./cover.jpg          # optional; omit → category glyph
coverAlt: "Alt text"        # required if cover set
draft: true                 # remove to publish
---
Body in Markdown. Co-locate images in the same folder; run them through
`python3 scripts/strip_resize.py IMG... --out src/content/projects/<slug>/`
first (bakes orientation, strips EXIF/GPS, resizes).
```

## Layout
```
src/config.ts            site identity, links, categories, analytics code
src/styles/tokens.css    design tokens (canonical + dormant themes)
src/content/projects/    one folder per project (md + images)
src/lib/                 calibration.js · spectra-viewer.js · wavelength-color.js
scripts/                 data pipeline, validation, image helper, token generator
public/data/spectra/     94 per-element NIST line files + finder index + manifest
```
Theme switching: set `data-theme="argon|cuvapor|kapton"` on `<html>` in
`src/layouts/Base.astro`, or edit `:root` in tokens.css.

## Tools data
